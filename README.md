# Learn to Hope

Open course material from [Pathway to Hope](https://pathwaytohope.ca), a Canadian
nonprofit that teaches young people aged 13 to 25 about mental health and
addiction. Everything we teach is meant to be read, copied, translated, and
improved by anyone.

## If you need support right now

This repository holds teaching material. Nobody monitors it, and it is not a way
to reach a counsellor. If you or someone else is in immediate danger, call
9-1-1.

| Service | Call | Text |
| --- | --- | --- |
| [9-8-8 Suicide Crisis Helpline](https://988.ca/) | 9-8-8 | Text 9-8-8 |
| [Kids Help Phone](https://kidshelpphone.ca/) | 1-800-668-6868 | Text CONNECT to 686868 |
| [ConnexOntario](https://www.connexontario.ca/) | 1-866-531-2600 | Text CONNEX to 247247 |
| [Hope for Wellness Help Line](https://www.hopeforwellness.ca/) | 1-855-242-3310 | Online chat |

All four are free and open 24 hours a day, every day. Hope for Wellness is for
Indigenous people across Canada. More options are listed at
[pathwaytohope.ca/support](https://pathwaytohope.ca/support).

Pathway to Hope provides education and community resources. We do not provide
counselling, diagnosis, treatment, or emergency response, and we are not a
crisis service.

## What is here today

Very little, and we would rather say so than have you dig for it. The repository
and the licence are in place, and there is a translation label set. That is
close to all of it. The curriculum itself is still being written, and it lands
here as each course is finished rather than in one release at the end.

If you came looking for a specific course and it is not here, it is not lost. It
is not written yet.

## How the material is organised

Everything sits under `academy/`, with course text in `academy/courses/`. The
layout is still settling and will change as real courses arrive, so treat any
path here as provisional rather than stable.

Courses are plain text so they stay readable, easy to diff, and easy to
translate without special tooling.

## Contributing

Corrections, translations, and adaptations are all welcome, and translations are
the most useful thing you can offer right now.

Open an issue before starting anything substantial so we can tell you whether it
is already in progress. For a small fix, a pull request on its own is fine.

Two things to know before you write. Material aimed at young people in distress
is held to a higher standard than most documentation, so claims about mental
health or addiction need a source we can check, and we will ask for one. And
nothing here should read as a substitute for professional care.
